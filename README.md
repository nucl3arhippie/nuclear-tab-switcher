
### üìñ Usage

#### Keyboard Shortcuts

| Shortcut     | Action                    |
| ------------ | ------------------------- |
| `ALT+Q`      | Open/close tab switcher   |
| `Tab`        | Navigate forward/backward |
| `Arrow Keys` | Navigate through grid     |
| `Enter`      | Switch to selected tab    |
| `Delete`     | Close selected tab        |
| `Escape`     | Close overlay             |

#### Mouse Controls

-   **Left Click** - Switch to tab
-   **Middle Click** - Close tab
-   **Hover** - Show close button

#### Customize Shortcut

1. Open `chrome://extensions/shortcuts`
2. Find "Enhanced Tab Switcher"
3. Set your preferred key combination

> **Note:** Chrome doesn't allow overriding `Ctrl+Tab`, so `ALT+Q` is used by default.

### üõ†Ô∏è Technical Details

-   **Manifest V3** - Latest Chrome extension API
-   **Vanilla JavaScript** - No external dependencies
-   **Lightweight** - ~15KB total size
-   **Privacy-focused** - No data collection, works offline

### ÔøΩ Troubleshooting

**Overlay doesn't appear:**

-   Ensure extension is enabled
-   Refresh the page (F5)
-   Check if shortcut conflicts with other extensions

**Doesn't work on some pages:**

-   Extensions can't run on browser internal pages (`chrome://`, `edge://`)
